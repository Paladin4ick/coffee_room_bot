# Команды бота

## Пользовательские команды

### Счёт и лидерборд

| Команда | Описание |
|---------|----------|
| `/score` | Показать свой текущий счёт |
| `/top` | Топ участников чата |
| `/history` | История последних начислений |

### Мут и защита

| Команда | Описание |
|---------|----------|
| `/mute @user <время>` | Замутить пользователя (списывает очки). Форматы: `30`, `30m`, `2h`, `1h30m` |
| `/selfmute <время>` | Замутить себя |
| `/protect` | Купить защиту от мута на 24 ч |

### Тег

| Команда | Описание |
|---------|----------|
| `/tag новый_тег` | Установить свой тег (за очки) |
| `/tag @user новый_тег` | Установить тег другому (за очки) |
| `/tag --clear` | Убрать свой тег |

### Экономика

| Команда | Описание |
|---------|----------|
| `/transfer @user N` | Перевести N очков пользователю |

### Игры

| Команда | Описание |
|---------|----------|
| `/bj <ставка>` | Блекджек против дилера |
| `/slots <ставка>` | Однорукий бандит |
| `/dice <ставка> <время>` | Кости: мультиплеер, победитель по наибольшему броску. Пример: `/dice 10 2m` |
| `/giveaway <призы> [время]` | Розыгрыш очков. Пример: `/giveaway 500 100 50 10m` |
| `/mutegiveaway <мут> <кол-во> <сбор>` | Мут-рулетка |

### AI

| Команда | Описание |
|---------|----------|
| `/llm <вопрос>` | Вопрос к языковой модели |
| `/search <запрос>` | Веб-поиск + AI-ответ со ссылками |

### Прочее

| Команда | Описание |
|---------|----------|
| `/help` | Интерактивная справка по разделам |
| `/limits` | Показать дневные лимиты |

## Административные команды

Префикс команд настраивается в `configs/config.yaml` → `admin.prefix` (по умолчанию `coffee`).

| Команда | Описание |
|---------|----------|
| `/<prefix>_add @user N` | Начислить N очков |
| `/<prefix>_sub @user N` | Списать N очков |
| `/<prefix>_set @user N` | Установить счёт |
| `/<prefix>_reset @user` | Сбросить счёт в 0 |
| `/<prefix>_op @user` | Назначить модератора |
| `/<prefix>_deop @user` | Снять модератора |
| `/amute @user <время>` | Мут без списания очков |
| `/unmute @user` | Снять мут |
| `/save @user` | Сохранить права администратора |
| `/restore @user` | Восстановить права |
| `/giveaway_end [id]` | Завершить розыгрыш |
| `/mutegiveaway_end [id]` | Завершить мут-рулетку |

## Формат времени

Везде, где нужно указать длительность, поддерживаются форматы:

| Пример | Значение |
|--------|----------|
| `30` | 30 минут |
| `30m` | 30 минут |
| `2h` | 2 часа |
| `1h30m` | 1 час 30 минут |
| `1d` | 1 день |
| `1d2h30m` | 1 день 2 часа 30 минут |
| `45s` | 45 секунд |
