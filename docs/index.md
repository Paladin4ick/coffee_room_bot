# Coffee Room Bot

Telegram-бот для геймификации групповых чатов.

Пользователи зарабатывают и теряют **кирчики ⭐** через emoji-реакции на сообщения, тратят их на игры, муты и теги.

## Технологии

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.12+ |
| Telegram | aiogram 3 |
| База данных | PostgreSQL 16 + asyncpg |
| Кэш / состояние игр | Redis 7 |
| Миграции | Flyway |
| DI | dishka |
| Пакетный менеджер | uv |
