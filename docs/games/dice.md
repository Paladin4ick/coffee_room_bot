# Игра в кости `/dice`

Мультиплеерная игра. Участники делают ставку, бот бросает за каждого встроенный Telegram-стикер 🎲, победитель по наибольшему значению забирает банк.

## Правила

1. Один участник запускает игру командой `/dice <ставка> <время>`
2. В течение указанного времени другие участники присоединяются кнопкой «Участвовать» — ставка списывается сразу
3. По истечении времени бот бросает кости за каждого (значения 1–6 от Telegram)
4. Игрок(и) с наибольшим значением делят банк поровну

## Команда

```
/dice <ставка> <время>
```

**Примеры:**
```
/dice 10 2m       # ставка 10, сбор 2 минуты
/dice 50 30s      # ставка 50, сбор 30 секунд
/dice 100 1h      # ставка 100, сбор 1 час
```

## Распределение приза

```
Банк = ставка × количество участников
Приз = банк ÷ количество победителей (округление — первому победителю)
```

Пример: 4 участника, ставка 25, двое выкинули по 6:
```
Банк = 25 × 4 = 100
Приз каждому победителю = 100 ÷ 2 = 50
```

## Ограничения

Настраиваются в `configs/config.yaml`:

```yaml
dice:
  min_bet: 1
  max_bet: 1000
  min_wait_seconds: 10    # минимум 10 секунд
  max_wait_seconds: 3600  # максимум 1 час
```

В одном чате может быть только одна активная игра одновременно.

## Технические детали

- Состояние игры хранится в PostgreSQL (`dice_games`, `dice_participants`)
- Фоновый цикл `dice_loop.py` каждые 5 секунд проверяет истёкшие игры
- Значения костей берутся из `message.dice.value` ответа Telegram на `send_dice(emoji="🎲")`
