# Блекджек

Классический блекджек против дилера прямо в Telegram-чате.

## Команда

```
/bj <ставка>
```

Примеры:

```
/bj 50
/bj 200
```

## Правила

1. Игрок делает ставку и получает две карты. Дилер получает две карты (одна скрыта).
2. Игрок выбирает:
   - **🃏 Ещё** — добрать карту
   - **🛑 Хватит** — остановиться, дилер открывает карты
3. Цель — набрать 21 или как можно ближе к 21, не превысив это число.
4. Дилер тянет карты, пока не наберёт 17 или больше.

### Стоимость карт

| Карта | Очки |
|-------|------|
| 2–10  | Номинал |
| J, Q, K | 10 |
| A | 11 (или 1, если иначе перебор) |

## Исходы

| Событие | Результат |
|---------|-----------|
| Блекджек (A + 10-карта) | Выигрыш × 2.5 от ставки |
| Выигрыш | Выигрыш × 2 от ставки |
| Дилер перебрал | Выигрыш × 2 от ставки |
| Ничья | Возврат ставки |
| Проигрыш | Потеря ставки |
| Перебор | Потеря ставки |

## Конфигурация

В `configs/config.yaml`:

```yaml
blackjack:
  min_bet: 1
  max_bet: 500
  max_games_per_window: 5   # максимум сессий на одного игрока
  window_hours: 1           # временное окно в часах
```

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `min_bet` | `1` | Минимальная ставка |
| `max_bet` | `500` | Максимальная ставка |
| `max_games_per_window` | `5` | Лимит игр за `window_hours` часов |
| `window_hours` | `1` | Размер временного окна (часы) |

## Технические детали

Состояние игры хранится в **Redis** (`RedisStore`), так как блекджек — это интерактивная сессия одного игрока. Ставка списывается в момент начала игры, выигрыш начисляется по итогу.

Логика колоды и подсчёт очков реализованы в `bot/application/blackjack_service.py`.
