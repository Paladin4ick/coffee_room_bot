# Слоты

Однорукий бандит с настраиваемыми символами и коэффициентами.

## Команда

```
/slots <ставка>
```

Примеры:

```
/slots 10
/slots 100
```

## Правила

Бот крутит 3 барабана. Результат определяется по совпадению символов:

| Результат | Условие | Выплата |
|-----------|---------|---------|
| **Джекпот** | Три одинаковых топ-символа (💎) | Ставка × множитель символа |
| **Выигрыш** | Три одинаковых любых символа | Ставка × множитель символа |
| **Почти** | Два одинаковых символа | Ставка × `near_miss_multiplier` (частичный возврат) |
| **Мимо** | Нет совпадений | Ставка потеряна |

## Символы по умолчанию

| Символ | Вес (частота) | Множитель |
|--------|--------------|-----------|
| 🍒 | 40 | × 2 |
| 🍋 | 30 | × 3 |
| 🍊 | 20 | × 5 |
| ⭐ | 8 | × 10 |
| 💎 | 2 | × 50 |

Чем меньше вес — тем реже выпадает символ.

## Конфигурация

Символы и параметры задаются в коде через `SlotsConfig` (в `bot/application/slots_service.py`):

```python
@dataclass
class SlotsConfig:
    min_bet: int = 1
    max_bet: int = 200
    reels: int = 3
    near_miss_multiplier: float = 0.45   # выплата при 2 совпадениях
    symbols: list[SlotSymbol] = ...      # список символов с весами и множителями
```

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `min_bet` | `1` | Минимальная ставка |
| `max_bet` | `200` | Максимальная ставка |
| `reels` | `3` | Количество барабанов |
| `near_miss_multiplier` | `0.45` | Доля ставки при двух совпадениях |

### Настройка RTP

Параметр `near_miss_multiplier` влияет на итоговый RTP (return to player):

| Значение | Приблизительный RTP |
|----------|---------------------|
| `0.45`  | ~121% |
| `0.40`  | ~118% |
| `0.0`   | ~95% |
| `-1.0`  | ~35% |

## Технические детали

- Состояние не хранится (одиночный запрос/ответ)
- Ставка списывается и выигрыш начисляется атомарно внутри одной транзакции
- Логика реализована в `bot/application/slots_service.py`
- Можно регистрировать `spin_guards` и `payout_modifiers` через декораторы для кастомного поведения
